    MenuItem(MenuItem),
    Submenu(Submenu),
    Predefined(PredefinedMenuItem),
    Check(CheckMenuItem),
    Icon(IconMenuItem),
}
Expand description

An enumeration of all available menu types, useful to match against the items returned from Menu::items or Submenu::items
