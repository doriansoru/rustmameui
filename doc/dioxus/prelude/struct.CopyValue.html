Struct CopyValue

Source 
pub struct CopyValue<T, S = UnsyncStorage>
where
    T: 'static,
    S: Storage<T>,
{ /* private fields */ }
Expand description

CopyValue is a wrapper around a value to make the value mutable and Copy.

It is internally backed by generational_box::GenerationalBox.
