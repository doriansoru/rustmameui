This module contains the Dioxus UI code for the application. It defines the main application component (App), handles state management using signals and memos, renders the tabbed interface for games, favourites, settings, and about sections, and manages user interactions like selecting games, launching games, adding/removing favourites, updating settings, and refreshing the game list.
